# AI-Career-Navigator

AI Career Navigator
Персональный карьерный ассистент для IT-специалистов.Помогает понять, куда расти, что учить и как попасть на желаемую позицию — на основе реальных вакансий и требований рынка.

О проекте
Мы создаём веб-платформу, которая поможет начинающим разработчикам (и не только) разобраться в карьерных возможностях и составить персональный план развития.
Сервис анализирует свежие вакансии с HH, Habr Career и других площадок, выявляет ключевые навыки для конкретных ролей и показывает, что именно нужно подтянуть, чтобы соответствовать позиции мечты.
Что умеет платформа:
Строит интерактивную карту навыков с зависимостями между ними
Определяет приоритеты в обучении
Рекомендует открытые курсы, статьи и книги
Позволяет пройти пробное AI-собеседование или тест по нужной теме
Генерирует примеры кода и задачи под ваш уровень и цель

Цель проекта
Создать персонального ассистента для IT-специалистов, который строит индивидуальную карту развития к желаемой цели на основе анализа рынка вакансий и требований к навыкам.

Технологический стек
Проект будет реализован с использованием следующих технологий:

Backend: Java, Spring Framework (+ Hibernate)
Брокер сообщений: Kafka
База данных: PostgreSQL
Frontend: Angular / Vue.js / React

Требования к участникам
Студенты будут работать в команде из 4 человек над созданием рабочего прототипа системы.
Работаем в отдельных ветках
Каждый участник создаёт свою ветку (например, name_branch). Никаких коммитов в main напрямую — только через Pull Request с ревью. Так мы избегаем конфликтов и контролируем качество.

Структура файлов:

Все исходники кода .java хранятся в папке src/.
Всё остальное — картинки, логотипы, скриншоты, документация — кладём в отдельную папку assets/ или docs/.
Не смешиваем код и медиа.

Еще пара правил:

Пишем понятные сообщения к коммитам.
Обсуждаем архитектурные решения перед тем, как писать код.
Если что-то не работает — не молчим, а пишем в чат. Мы команда — вместе решаем.

Ожидаемый образовательный результат
В ходе работы над проектом студенты получат практические навыки в:

Командной разработке.
Управлении проектами.
Управлении базами данных.
Веб-разработке.
